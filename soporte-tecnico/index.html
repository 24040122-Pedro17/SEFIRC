Sistema de soporte técnico

Portal interno para que el personal de la Universidad Tecnológica de Calvillo solicite atención al área de TI — desde incidentes hasta capacitaciones.

Ver en GitHub Pull request #1

Tipos de solicitud

Incidente

Algo no funciona: equipo, internet o una app dejó de responder.

Asesoría técnica

Tienes una duda o necesitas orientación, pero nada está fallando.

Capacitación

Quieres aprender a usar una herramienta o sistema desde cero.

Reunión

Necesitas coordinar una junta con el equipo de soporte.

Flujo de una solicitud

Login con AD → Seleccionar categoría → Llenar formulario → Folio generado → Seguimiento
